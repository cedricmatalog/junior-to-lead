# Contributing

Thanks for helping improve the curriculum.

## How to Contribute
- Review `AGENTS.md` for repository guidelines and conventions.
- Keep module structure consistent (5-8 chapters, narrative tone, and required sections).
- Match the existing tone and formatting in each module.

## Pull Requests
- Use Conventional Commits (for example, `feat: add module`, `fix: correct example`).
- Include a short description and testing notes (if any).

## Tests
No automated test tooling is configured yet.
