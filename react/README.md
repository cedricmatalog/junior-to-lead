# React Developer Curriculum: Junior to Lead

A comprehensive learning path for React developers progressing from junior to lead-level positions.

## Overview

This curriculum covers both **technical React skills** and **leadership competencies** through a linear progression across four levels. Each level builds on the previous, with clear milestones marking your growth.

## Curriculum Structure

| Level | Focus | Modules | Target |
|-------|-------|---------|--------|
| [01 - Junior](01-junior/) | Foundation | 13 | Solid fundamentals, ship features independently |
| [02 - Mid-Level](02-mid-level/) | Proficiency | 17 | Own complex features, production-grade code |
| [03 - Senior](03-senior/) | Expertise | 21 | Architectural decisions, mentor others |
| [04 - Lead](04-lead/) | Leadership | 12 | Technical direction, grow teams |

**Total: 63 modules across 4 levels**

## How to Use This Curriculum

1. **Assess your current level** - Read each level's README to understand where you are
2. **Work through modules sequentially** - Each builds on previous knowledge
3. **Practice with real projects** - Apply concepts to your daily work
4. **Track your progress** - Check off completed modules
5. **Move up when ready** - Complete the milestone before advancing

## Prerequisites

This curriculum assumes you are a **working junior React developer** with:
- Basic HTML, CSS, and JavaScript knowledge
- Some React experience (built basic apps)
- Familiarity with npm/yarn and modern JS tooling
